## Overview

This project explored the implementation of professional game audio workflows using FMOD integrated with Unity.

Working as part of a multidisciplinary team, the project combined audio design and programming, allowing audio students to create assets while developers implemented the technical systems required to bring those assets into the game environment.

## Challenges and Solutions

### Integrating Audio Systems into Gameplay

A key challenge was ensuring that audio events responded correctly to gameplay situations while remaining easy to manage and extend.

This was addressed by creating event-driven systems that enabled audio triggers to be flexibly connected to gameplay interactions.

### Cross-Discipline Collaboration

Working with audio students required translating creative audio requirements into technical implementations.

Regular communication to ensure that the intended audio experience was successfully realised within the game environment.

Helping audio students understand how to use Unity and teaching them to write basic code so they can understand.
